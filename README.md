# simple-python-calculator
Простой калькулятор на Python с обработкой ошибок

<Консольное приложение для выполнения базовых арифметических операций: сложение, вычитание, умножение, деление.>

# Описание
Этот проект демонстрирует:
- Ввод данных от пользователя с обработкой ошибок.
- Работу с числами (поддержка целых и дробных значений).
- Структурирование кода через функции.
- Защиту от деления на ноль.
- Комментарии для читаемости кода.


# Функционал
- **Сложение** (`+`)
- **Вычитание** (`-`)
- **Умножение** (`*`)
- **Деление** (`/`) с проверкой на ноль
- Обработка некорректного ввода (буквы, символы)


# Как запустить
1. Клонируйте репозиторий:
   
   https://github.com/MaDevMax/simple-python-calculator.git
 
 3. Перейдите в папку проекта:

cd simple-calculator

Запустите скрипт:

python calculator.py

Пример работы
Простой калькулятор (поддерживает +, -, *, /)
Введите первое число: 10
Введите второе число: 3

Результаты:
Результат для +: 13.0
Результат для -: 7.0
Результат для *: 30.0
Результат для /: 3.3333333333333335
